# Secure Smart Landing Page — SPEC

## Goal
Create a premium B2B public landing page for Secure Smart that builds trust with professional buyers while keeping prices gated until registration.

## Users
- Installers / integrators
- Security teams
- Kibbutzim, moshavim and gated communities
- Senior living / community infrastructure buyers
- Resellers and professional catalogue buyers

## Requirements
- Public marketing page with no visible prices.
- Show a selected launch catalogue of 10–12 flagship products.
- Emphasize professional infrastructure: Ubiquiti, MikroTik, Wi-Fi, switching, cameras, NVR, access and 60GHz backhaul.
- Clear registration CTA to unlock catalogue/pricing.
- Explain package/carton logic and 5% full-carton discount after registration.
- Work well on desktop and mobile.
- Deploy to GitHub Pages and Netlify.

## Out of scope for this phase
- Real backend registration handling.
- Live CRM/email integration.
- Private authenticated catalogue.
- Payment/cart checkout.
